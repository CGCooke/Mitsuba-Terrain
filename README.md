# Mitsuba-Terrain


## How do I get started?

1. Install the environment.

```sh
conda env create -f environment.yml
```
2. Activate the environment
```sh
conda activate Mitsuba-Terrain
```

4. Launch Jupyter
```sh
jupyter notebook
```

5. Run 'Simple Render.ipynb'

6. Enjoy your rendering.
